Оценка по трём точкам

Метод позволяет учесть оптимистичную, реалистичную и пессимистичную оценку задачи.
  1. Оптимистичная оценка — время при идеальном ходе работ.
  2. Реалистичная — сколько задача занимает обычно с учётом рисков.
  3. Пессимистичная — сколько может потребоваться в худшем случае.
Итог вычисляется по формуле (O + 4R + P) / 6.

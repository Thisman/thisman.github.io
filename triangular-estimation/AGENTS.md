# AGENTS.md — Треугольная оценка

Калькулятор для оценки задачи по методу PERT. Даёт взвешенный результат на основе трёх точечных оценок.

## Что делает

Пользователь вводит три числа:

- **O** — оптимистичная оценка (лучший сценарий)
- **R** — реалистичная оценка (наиболее вероятный сценарий)
- **P** — пессимистичная оценка (худший сценарий)

Выбирает единицу измерения: часы, дни или недели. Нажимает «Рассчитать» — получает одно число. Изменение единицы измерения тоже запускает пересчёт.

## Логика

Формула: `⌈(O + 4R + P) / 6⌉`

Реалистичная оценка весит в четыре раза больше крайних значений. Результат округляется вверх до целого.

Перед расчётом проверяется условие O ≤ R ≤ P. Если порядок нарушен — показывается ошибка с пояснением, какое именно значение некорректно.

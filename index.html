Сайт позволяет посчитать количество сторипоинтов для оценки задачи.

Оценка строится из трех шагов:
  1. Оценить кол-во информации которое известно о задаче и её решении uncertainty
  2. Оценить сложность решения задачи. Насколько легко или трудно реализовать задачу, когда о ней всё известно? complexity
  3. Оценить объем решения задачи. Сколько обязательных шагов подразумевает решение задачи? effort

Чтобы получить оценку задачи, выбери ниже для каждого критерия значение.
Какая неопределенность у задачи? uncertainty
Какая сложность у решения задачи? complexity
Какой объем у решения задачи? effort

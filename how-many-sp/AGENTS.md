# AGENTS.md — Сколько сториков?

Калькулятор для оценки задачи в сторипоинтах. Помогает команде приходить к воспроизводимой оценке, убирая субъективность.

## Что делает

Пользователь выбирает три параметра задачи:

- **Uncertainty** — насколько понятна задача и её решение (low / medium / high)
- **Complexity** — насколько сложно реализовать при полном понимании (low / medium / high)
- **Effort** — сколько обязательных шагов требует решение (low / medium / high)

По комбинации трёх параметров из матрицы достаётся число по шкале Фибоначчи (1, 2, 3, 5, 8, 13, 21). Результат обновляется сразу при изменении любого параметра.

## Логика

Матрица `SP_MATRIX[uncertainty][complexity][effort]` задана статически. Все 27 комбинаций разобраны вручную.

Цвет результата:
- зелёный — 1–4 SP (задача понятная и небольшая)
- жёлтый — 5–7 SP (стоит подумать о декомпозиции)
- красный — 8 и выше SP (рекомендуется разбить)

При высокой оценке появляется текстовая подсказка с рекомендацией.

# AGENTS.md — The Maze (Dual Grid Puzzle)

Головоломка, где одним нажатием клавиши управляются сразу несколько персонажей на разных картах. Нужно привести всех к цели за точное число шагов.

## Что делает

Экран разбит на 1–4 карты. На каждой карте есть персонаж (A, B, C, D) и целевая клетка. Стрелки или WASD двигают всех персонажей одновременно в одном направлении. Победа — все персонажи стоят на своих целях. Количество шагов строго ограничено: нужно уложиться ровно в T ходов.

## Типы клеток

- **Стена** — непроходима.
- **Лёд** — персонаж скользит до первого препятствия, не останавливаясь на промежуточных клетках.
- **Пружина** — после входа персонаж делает ещё один шаг в том же направлении (если клетка свободна).
- **Яма** — мгновенное поражение при попадании.
- **Рассыпающаяся клетка** — после первого прохода становится стеной.
- **Болото** — персонаж застревает на один ход: следующая команда его не двигает.
- **Стрелка** — пропускает персонажа только с определённого направления; с остальных — блокирует.
- **Телепорт** — пара клеток вход/выход; персонаж появляется на выходе.
- **Кнопка и дверь** — пока персонаж стоит на кнопке, соответствующая дверь открыта.

## Логика хода

При каждом действии для каждого персонажа последовательно применяются правила: проверяется достижимость следующей клетки, затем особые эффекты (лёд, пружина, телепорт). Болото пропускает ход персонажа целиком. После всех перемещений проверяются условия победы и поражения.

Кнопка и дверь работают динамически: дверь открыта только пока кто-то стоит на кнопке — это учитывается при каждом шаге.

## Генерация уровней

Уровни генерируются процедурно по зерну. Сложность нарастает по уровням: растёт размер карты (от 4×4 до 8×8), число карт (от 1 до 4), бюджет шагов (от 4 до 22), вводятся новые типы клеток.

Генератор прокладывает случайный путь из стартовой позиции длиной T, расставляя особые клетки с заданной вероятностью на достижимых позициях. Затем BFS-солвер проверяет, что уровень разрешим — если нет, генерируется заново.

## Встроенный солвер

`window.solve()` запускает BFS по пространству состояний (позиции всех персонажей + рассыпавшиеся клетки + статус болота). Возвращает последовательность ходов до победы или сообщает, что решения нет.

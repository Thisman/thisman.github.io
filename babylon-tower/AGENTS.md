# AGENTS.md — Вавилонская башня

3D-головоломка на пятиугольной призме. Цель — расставить фишки так, чтобы на каждой грани оказались фишки одного цвета.

## Что делает

Призма состоит из 5 слоёв × 5 граней = 25 клеток. На призме 20 фишек четырёх цветов (по 4 каждого) и 5 пустых клеток. Пользователь совершает ходы двух типов, пока каждая грань не станет монохромной.

## Ходы

**Вращение слоя** — все 5 фишек одного горизонтального слоя сдвигаются на одну позицию по кругу (по часовой или против). Для каждого из 5 слоёв доступны обе стороны.

**Скольжение** — выбранная фишка меняется местами с соседней пустой клеткой на той же грани. Движение только по вертикали внутри грани.

## Логика

Состояние хранится в двумерном массиве `state[layer][face]`. Значения — 0 (пусто) или номер цвета (1–4).

При вращении слоя все значения строки циклически сдвигаются: `newFace = (face ± 1) % 5`. Инвариант — ровно 5 пустых клеток сохраняется всегда.

При выборе фишки подсвечиваются допустимые цели для скольжения: только пустые клетки той же грани, соседние по вертикали. После хода позиции меняются местами.

Проверка победы: каждая грань должна содержать только фишки одного цвета (грань 0 — пустая).

Скрамблинг выполняется 30 случайными ходами, исключая немедленно обратный ход. Undo хранит историю состояний.
